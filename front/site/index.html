🌊 Stream Run 1

Volte no tempo. Lute contra a poluição. Salve o planeta.

⚙️ Sobre o Jogo

Stream Run 1 é um jogo de plataforma 2D criado como parte de um projeto acadêmico do curso de Desenvolvimento Web. Desenvolvido com Python e rodando no VS Code, o jogo combina ação retrô com uma forte mensagem ecológica.

Nele, você assume o papel de Gusion, um herói que viaja no tempo para enfrentar corporações malignas e seus capangas que estão destruindo o planeta com poluição e lixo. Em cada fase, você enfrentará diferentes vilões — como o Orc, o Gorila de petróleo e o Macaco robótico — cada um representando formas de degradação ambiental.

A missão é clara: derrotar os inimigos e impedir que a destruição avance. Com visuais retrô, trilha sonora envolvente e jogabilidade fluida, Stream Run 1 prova que jogos podem entreter e educar ao mesmo tempo.

🧑‍🚀 Personagem Principal

Gusion é um viajante do tempo encarregado de impedir que corporações poluentes destruam o planeta.

Corajoso, ágil e com senso de justiça, ele luta para reverter o curso da destruição ambiental.

Vilões

Orc – Capanga envolto em fumaça tóxica que contamina tudo por onde passa.

Gorila – Criatura feita de petróleo que polui rios e florestas.

Macaco – Robô de lixo plástico que devora materiais recicláveis.

💻 Como Jogar

Stream Run 1 é um jogo feito em Python que roda diretamente no terminal. Siga os passos abaixo para jogar:

  1. Instale o Python em seu computador (python.org).
  2. Baixe os arquivos necessários aqui.
  3. Abra o Visual Studio Code (VS Code).
  4. Abra a pasta do jogo no VS Code.
  5. Abra o terminal (atalho: Ctrl + `).
  6. Instale as bibliotecas necessárias com pip install
  7. Direto no terminal
  8. Exeute o Codigo

Agora é só jogar e ajudar Gusion a salvar o planeta! 🌍
